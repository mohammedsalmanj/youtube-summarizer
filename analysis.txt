| Aspect                  | Phase 1 (V1)                 | Phase 2 (V2)                                                        | Improvement / Analysis                                                                                                              |
| ----------------------- | ---------------------------- | ------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| **Utils Used**          | `utilsold`                   | `utils` + `remove_repeated_lines`                                   | Added preprocessing to clean transcripts before summarization.                                                                      |
| **Transcript Handling** | Raw transcript directly used | Transcript cleaned to remove repeated words/lines                   | Reduces redundancy and messy summaries.                                                                                             |

| **FAISS Index**         | Created from raw chunks      | Created from cleaned chunks                                         | Q\&A is more accurate and less redundant.                                                                                           |
| **Gradio UI**           | 8-line summary box           | 10-line summary box                                                 | Slightly more space for clearer output.                                                                                             |
| **Output Quality**      | Repetitive, less concise     | Clear, non-redundant bullet points                                  | Phase 2 output is professional and usable.                                                                                          |
| **Overall**             | Quick prototype              | Preprocessing + better prompt + deduplication                       | Phase 2 reduces errors, improves clarity, sets stage for further improvements like semantic chunking or hierarchical summarization. |
